# Sprint_4 - Автоматизация Яндекс Самоката

Проект в рамках программы обучения "Инженер по тестированию:
от новичка до автоматизатора"

Автоматизация тестирования сайта Яндекс Самоката: проверка заполнения формы заказа, 
ответов на важные вопросы, переходы по логотипам, отслеживание статуса заказа

## Используемые технологии

| Технология      | Версия         | Применение                                                                                  |
|-----------------|----------------|---------------------------------------------------------------------------------------------|
| Java            | 11             | Язык, на котором писались автотесты проекта                                                 |
| Selenium        | 3.141.59       | Фреймворк автоматизации веб-браузеров, используется для проведения автотестов               |
| JUnit           | 4.12           | Фреймворк программирования автоматизированного тестирования для языка программирования Java |
| Maven           | 4.0.0          | Фреймворк автоматизации сборки проектов, через который устанавливались указанные технологии |
| Chromium        | 144.0.7559.109 | Браузер, на котором проводилось тестирование                                                |
| Mozilla Firefox | 140.7          | Бразуер, на котором проводилось тестирование                                                |
